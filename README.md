# AI Website Builder — Website Builder for Small Businesses

Build a professional website for your business in 5 minutes. No coding required.

## Supported Business Types
- Barbershop
- Salon / Beauty
- Restaurant / Food
- Consultant / Coach
- Online Store
- Mechanic / Auto
- Tutor / Educator
- Church / Ministry
- Custom

## Features
- Guided wizard with smart defaults
- Live website preview with responsive testing
- Mini CRM: appointments, clients, stats
- Multiple design styles and brand colors
- Mobile-friendly generated websites

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- lucide-react
- motion (Framer Motion)
- React Router

## Run Locally
npm install
npm run dev
